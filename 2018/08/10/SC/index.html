Stochastic Circuits Optimization

Posted on 2018-08-10 | In projects

This is an research work, and here is the link to the paper:
Simultaneous Area and Delay Optimization for Stochastic Circuits by D Flip-flop Insertion

Problem definition

Stochastic Computing

This is an computing technique using digital circuits. The idea is to encode real values through the probabilities of ones on bit streams, and then performs arithmetic computation.

Bit Stream Representation of Real Values

For example, the bit stream 10011010 can represent 0.5, or 1/2 since there are 4 ones in the 8-bit-long bit stream.

Arithmetic Computation

AND gate can perform multiplication a·b while OR gate can perform a + b - a·b.
For example, given two bit streams, a = 11010111 represents 6/8 and b = 11001010 represents 4/8. Let them go through an AND gate bit by bit, the output will be c = 11000010 represents 3/8, where c = a·b.

Challenges

We want to reduce the computation time and large hardware overhead to generate independent stochastic bit streams.

It seems that this can be addressed by sharing the stochastic number generator (SNG). However, this will lead to the maximal correlation of the generated stochastic bit streams. If so, consider the example of AND gate above, a = 11010111 and b = 11000110, still represents 6/8 and 4/8. However, the output will become c = 11000110 represents 4/8, or 1/2. In this situation, the AND gate cannot perform multiplication any longer, but give out the result of min{a, b}.

Idea

To ensure that the stochastic circuit can give out the expected result, the input stochastic bit streams have to be decorrelated. We can fulfill this target through insertion of D-Flip-Flops (DFFs).

The insertion of DFFs can potentially break a critical path, and hence, reduce the circuit delay.

Further, we hope that under the premise of functional correctness, least number of DFFs should be inserted to reduce area overhead.

Therefore, the point is how to ensure that with least number of DFFs, the circuit still has an equvalent functionality. For a given circuit, we list a set of linear equalities and inequalities that guarantees the functional equivalence, based on the method to obtain the stochastic function of a stochastic circuit with DFF inserted. And the solution is calculated with the help of integer linear programming (ILP).
